# SessionGoals

Parkour and freerunning progression planner built with SvelteKit and Supabase.

## Vision

SessionGoals helps athletes:

- save moves they want to learn
- save spot-based goals
- collect tutorials and inspiration
- organize challenge lists
- discover matching spots

## Planned stack

- SvelteKit
- Supabase
- TypeScript
- Tailwind CSS
