# Design Context — SessionGoals

## Target Audience
Both beginners discovering parkour and dedicated practitioners managing ongoing training across spots and move lists.

## Use Cases
- Track personal goals (moves, spots, challenges) through a progression pipeline
- Plan session locations based on active goals assigned to spots
- Discover and follow community lists; track progress against them
- Log inspiration for future training

## Brand Personality / Tone
Athletic + motivating. Energetic, bold, like a physical training journal. Feels earned, not corporate. The interface should feel like something a real traceur actually wants to use — not a generic task manager with sport branding applied.

## Technical Context
SvelteKit + TypeScript, CSS custom properties, light theme, OKLCH color palette. Barlow Condensed (display) + Hanken Grotesk (body).
